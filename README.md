# ProntoLogin
Jetpack Compose UI app demonstrating using basic Jetpack Compose components to create login screen

This Single screen app implements this [Figma](https://www.figma.com/file/iRQ8twpwVgSwuAhKS8paGG/Pronto-Login-Screen?node-id=1%3A715&t=OuJL4nY79zBFeywX-1) mock login screen.

This app is part of my Jetpack Compose tutorial series called Master Jetpack Compose in 100 Days.
